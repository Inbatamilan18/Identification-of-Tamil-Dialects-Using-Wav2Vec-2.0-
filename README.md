# Tamil Dialect Identification Using Wav2Vec 2.0

This project identifies Tamil dialects using deep learning and Wav2Vec 2.0.

## Features
- Tamil dialect classification
- Audio processing
- Machine learning based prediction
- Visualization using confusion matrix and PCA clusters

## Technologies Used
- Python
- Jupyter Notebook
- Wav2Vec 2.0
- Pandas
- Scikit-learn
- Matplotlib

## Files Included
- Tamil_Dialect_Identification.ipynb
- confusion_matrix.png
- pca_clusters.png
- dialect_predictions.xlsx
